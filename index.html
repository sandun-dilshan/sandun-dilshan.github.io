Research & Publications

Smart Motorcycle Security System with Multi-Modal Authentication and IoT Integration

Designed and developed a smart motorcycle security system integrating biometric authentication, mobile application control, and GPS tracking. The system enables secure vehicle access using fingerprint verification, with additional control through a Flutter-based mobile application and Bluetooth communication.

The solution includes real-time location tracking, SMS-based fallback communication, and ignition control, ensuring functionality even in low-connectivity environments. This project demonstrates a practical, low-cost IoT-based security solution suitable for real-world deployment.
